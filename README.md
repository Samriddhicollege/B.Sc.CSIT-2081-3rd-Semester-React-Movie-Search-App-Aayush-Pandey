## Project Overview
> This project is a modern web-based movie discovery application built using React and Vite.  
> It allows users to explore trending movies, upcoming releases, and films currently playing in cinemas.  
> Users can search for movies and watch trailers directly within the application.  
> The system provides a smooth and responsive user interface with carousel-based navigation using Swiper.  
> The main goal of this project is to create an engaging platform for browsing and discovering movie content efficiently.
